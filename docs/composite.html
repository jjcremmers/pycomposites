class TransverseIsotropic(E: float | list[float], nu12: float, G12: float = 0.0, alpha: float | list[float] = 0.0, rho: float = 0.0)[source]

Bases: object

A class to represent a transversely isotropic material model.

This class models materials that exhibit isotropic properties in a plane perpendicular to a single axis of symmetry and anisotropic behavior along the axis of symmetry.

setAlpha(alpha: float | list[float]) → None[source]

Set the thermal expansion coefficients (alpha) for the material.

Parameters:

alpha (float or list[float]) – Thermal expansion coefficients. Can be a single float value or a list of one or two float values. If a single value is provided, it is assigned to both alpha1 and alpha2. If a list of two values is provided, the first value is assigned to alpha1 and the second to alpha2.

Raises:

ValueError – If the list has more than two elements.

setFailureProperties(F: list[float], Gfrac: float | list[float] = 0.0, alpha0deg: float = 53.0) → None[source]

Set the failure properties of the material.

Parameters:

  • F (list[float]) –

    A list containing the main failure parameters: F = [Xt, Xc, Yt, Yc, Sl] where:

    • Xt: Longitudinal tensile strength

    • Xc: Longitudinal compressive strength

    • Yt: Transverse tensile strength

    • Yc: Transverse compressive strength

    • Sl: Transverse shear strength

    For the Tsai-Wu failure criterion, f12 can be provided as the 6th parameter: F = [Xt, Xc, Yt, Yc, Sl, f12].

  • Gfrac (float or list[float], optional) – Fracture toughness. Can be a single value or a list of two values [GIc, GIIc]. Defaults to 0.

  • alpha0deg (float, optional) – Alpha0 in degrees, needed for the Larc03 model. Defaults to 53.0.

Raises:

ValueError – If F has an invalid length.
